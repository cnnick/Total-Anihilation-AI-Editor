//
//  Details about this version:
//
//    a) Recommended for maps with STARTING POSITIONS JOINED BY LAND crossable by AI.  DEFAULT is intended for those maps
//       unable to support ships but feasible for hovercraft to play a lesser role than in DEFAULThover.  For example, a
//       smaller quantity of underwater metal deposits or limited benefits from crossing ponds could justify building fewer
//       and lower priority hovercraft.
//
//       There are a variety of Escalation AI profiles for land having similar recommendations with subtle differences
//       between them, including Urban, DEFAULT, DEFAULThover and DEFAULTships.  A flowchart showing the simplified process
//       of choosing aiprofile for any map and helping clarify the subtleties among various DEFAULT hybrids is available at:
//
//         https://www.tauniverse.com/forum/showthread.php?t=46159
//
//    b) Does NOT build ships or submerged structures other than Underwater Metal/Moho/Mantle Extractors
//
//    c) Does NOT build Escalation AI cheat units ARMCHEAT, ARMUWCHEAT, CORCHEAT, CORUWCHEAT.  However, Escalation v9.5+ have
//       built-in AI economy multipliers of 2x and 4x, respectively, for Medium and Hard difficulty.  Easy is noncheating.
//
//    d) Builds long range artillery (terrain permitting) using following limits according to game difficulty setting:
//
//                                             Easy    Medium    Hard
//         Arm Big Bertha / Core Intimidator     0        4        8
//         Arm Vulcan / Core Buzzsaw             0        4        8  <-- Requires optional unit customization (details below)
//         Arm Armageddon / Core Dictator        0        2        4
//
//         The above nonzero artillery values can be exceeded because game engine does not count incomplete buildings when
//         tallying unit quantities throughout a game.  That quirk is a consideration in chosen values, above.
//
//         AI ability to trigger unit upgrades from Big Bertha / Intimidator to become Vulcan / Buzzsaw requires unit custom-
//         ization via optional addon files.  Basically, unit fbi data file is customized to enable building upgraded version.
//         This AI feature is unavailable within Escalation releases because structures morphing to become other structures
//         have lead to some network data synchronization issues between players during online games.  That issue does not
//         exist during Skirmish.  If you play both Skirmish and online, it is highly recommended having two separate install-
//         ations of Escalation to isolate Skirmish setup from online.  Optional addon AI unit upgrade files are available at:
//           (for Escalation v9.6/9.7)   https://www.tauniverse.com/forum/showpost.php?p=770812&postcount=8
//         Since, Big Bertha / Intimidator units did not change between v9.6 and 9.7, their AI unit upgrade files at above
//         link are also accurate for v9.7.  Additional features within that addon pack allow player to optionally disable
//         AI long range artillery.  Each fbi file defines a specific unit allowing artillery units to be enabled or disabled
//         on an individual basis.  Player can choose to disable artillery unit(s) for only one faction if that is desirable.

//
//  Additional tidbits:
//
//    An AI controlled mobile combat unit can temporarily stall in Standby mode on water maps when nearest enemy target is
//    not targetable by the combat unit's weapon(s).  Standby mode can linger until non-targetable unit is destroyed or a
//    targetable enemy unit moves closer.  Some examples of naval stall include:
//
//    - mobile land unit fixating on a submerged unit (Wotan and Admiral_94 have improved this within Escalation as of 2021)
//    - certain hovercraft (without depth charge/torpedo) fixating on a submerged unit
//    - submerged amphibious unit (without torpedo) fixating on a floating unit
//    - certain ships (without depth charge/torpedo) fixating on a submerged unit
//    - submarine fixating on a hovercraft unit
//
//    Naval stall has existed in varying degrees since Total Annihilation v3.1c around 1998.  Its frequency increases as
//    amount of submerged units increases.  The following list shows some significant plateaus.
//
//      Total Annhilation Version      Number of Submerged Units
//
//      Total Annihilation v3.1c          5
//      Core Contingency expansion       15
//      Escalation Beta v8.9.0           23
//
//    The naval Standby mode stall is related to game engine's limited handling of AI units.  Its one-track mind causes an AI
//    mobile combat unit to focus on nearest enemy target even when unit's weapon(s) can not deal with the target.  Naval
//    stall frequency would improve if game engine allowed AI unit cycling to second (third, fourth or fifth) nearest target
//    before resorting to Standby mode.
